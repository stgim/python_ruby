`OR`
```
OR은  두 입력 값중에서 하나라도 True 일 경우 True 출력
```
| 입력 | 기호 | 입력 | 출력 |
|---|:---|---|---|---:|
|True| or|	True|	True|
True	|or|	False|	True|
False	|or|	True|	True|
|False|	or|	False|	False|

`AND`
```
AND는 두 입력 값중에서 두 입력 모두 True 일 경우 True 출력
```
| 입력 | 기호 | 입력 | 출력 |
|---|:---|---|---|---:|
|True |and 	|True|	True|
|True	|and|	False|	False|
|False	|and|	True|	Flase|
|False	|and|	False|	False|

`NoT`
```
Not은 영상 수업이 없습니다.
```
| 기호 | 입력 | 입력 |
|---|:---|---:|
| NOT | True | False |
| NOT | False | True |
